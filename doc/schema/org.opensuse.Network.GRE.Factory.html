string newDevice(string name, dict config)

Argument config is mapped to element <gre> of the device document. It is a dict-based type that provides the following members:

  • device (string). This is a string
  • local-address (ipv4-address).
  • remote-address (ipv4-address).
  • ttl (uint16).
  • tos (uint16).
  • pmtudisc (boolean).
  • flags (builtin-gre-flag-mask).
  • ikey (ipv4-address).
  • okey (ipv4-address).
