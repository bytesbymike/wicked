string newDevice(string name, dict config)

Argument config is mapped to element <ppp> of the device document. It is a dict-based type that provides the following members:

  • mode (unknown). This is an unknown
  • debug (boolean).
  • demand (boolean).
  • persist (boolean).
  • idle (uint32).
  • maxfail (uint32).
  • holdoff (uint32).
  • multilink (boolean).
  • endpoint (string).
  • dns (ppp:dns-config).
  • defaultroute (boolean).
  • auth (ppp:auth-config).
  • ipv4 (ppp:ipv4-config).
  • ipv6 (ppp:ipv6-config).
