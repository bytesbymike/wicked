org.opensuse.Network.MACVLAN

Synopsis

This link-layer service is provided by all MACVLAN devices. Since there isn't a lot to tune about a MACVLAN once it has been created, this DBus interface doesn't offer an own changeDevice() method (yet).

MACVLAN devices can be created through the org.opensuse.Network.MACVLAN.Factory factory interface.
